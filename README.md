# Job Board API

A simple REST API for managing job postings. Built with **Node.js**, **Express**, and **MongoDB (Atlas)**.

---

## Features
- Create, read, update, and delete job postings
- Data stored in MongoDB Atlas
- RESTful endpoints with JSON responses

---

## Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- dotenv for environment variables
